# IndustrialCraft-Reborn
Conversão não oficial do IndustrialCraft para Minecraft Java 26.1.2, feita por fã para manter vivo um mod que marcou muitos jogadores.
Unofficial IndustrialCraft conversion for Minecraft Java 26.1.2, fan-made to keep alive a mod that meant a lot to many players.

> Este projeto é uma conversão não oficial/fan-made do IndustrialCraft para Minecraft Java 26.1.2.
> This is an unofficial fan-made conversion of IndustrialCraft for Minecraft Java 26.1.2.

Este projeto foi criado por carinho ao mod original, que marcou muitas jogatinas com amigos. O objetivo é manter a experiência viva em versões mais recentes do Minecraft.
This project was created out of appreciation for the original mod, which was part of many great moments with friends. The goal is to keep the experience alive on newer Minecraft versions.
